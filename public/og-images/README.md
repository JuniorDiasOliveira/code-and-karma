# Open Graph Images

This folder contains Open Graph (OG) images for social media previews.

## Image Specifications

### Required Images:
- `og-default.jpg` - Default fallback image
- `og-home.jpg` - Homepage image
- `og-about.jpg` - About page image
- Individual post images: `{post-slug}.jpg`

### Dimensions:
- **Recommended size**: 1200x630px
- **Aspect ratio**: 1.91:1
- **File format**: JPG or PNG
- **File size**: Under 1MB (ideally under 300KB)

### Design Guidelines:
- Use your brand colors (neon theme)
- Include the blog title "Code & Karma"
- For blog posts, include the post title
- Keep text readable at small sizes
- Consider dark/light theme variations

### Tools for Creating OG Images:
1. **Canva** - Templates available
2. **Figma** - Design custom templates
3. **OG Image Generator** - Automated solutions
4. **Bannerbear** - API-based generation

### Dynamic OG Images (Future):
Consider implementing dynamic OG image generation using:
- Vercel's @vercel/og
- Playwright for screenshot generation
- Canvas API for programmatic generation
